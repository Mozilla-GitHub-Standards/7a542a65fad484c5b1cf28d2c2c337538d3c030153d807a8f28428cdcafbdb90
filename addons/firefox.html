Mozilla : Web Applications for Firefox

Web Applications are applications that run on any device, and can be distributed through any store or directly by the developer.

The experimental Web Apps extension allows you to easily install and manage Web Applications in Firefox and aims to provide a tightly integrated app experience.

 (Requires Firefox 4 beta or better)

Launch

This feature adds an "Installed Applications" toolbar button to Firefox. You can launch applications directly from the toolbar by clicking the button and then selecting the application you want to launch in the popup window.

Applications will automatically be started in an App Tab, and are recognized as 'Web Applications' across sessions.

Note: In this version, the list of installed applications used by the extension is not synchronized with the default (non-extension-based) list. After installing the extension, you will need to re-install any applications you already have.

Discover

We've added experimental support for discovering which web sites provide an installable Web App. The toolbar icon will glow and provide you with a prompt if the extension detects the presence of an installable application.

Note: Not all web applications support discovery, as we have not yet standardized the mechanism.

Manage

In addition to being able to manage your applications from a dashboard of your choice, you can also manage them via 'about:apps'. Application manifests are also made available for inspection.

Extend

The Web Application extension for Firefox is the ideal vehicle to test deeper desktop integration for Web Apps. To this end, we have added experimental support for propagating media key events (on Windows) to installed applications. With a simple one-line script, you can now control your favorite web-based music player via keyboard shortcuts even if Firefox is not the active window!

We are also working on many more experimental features to enhance the Web App experience, such as integrating with Firefox Sync to seamlessly synchronize your installed apps across multiple devices.

This alpha extension is continuously evolving as we experiment with new ways to integrate the web app experience with your desktop. We welcome your contributions, either as code or comments in our mailing list.
