Mozilla : Firefox Web Application Extension

Web Applications are applications that run on any device, and can be distributed through any store or directly by the developer.

Firefox Web Application Extension

This experimental extension adds an "installed applications" toolbar button to Firefox.

You can launch applications directly from the toolbar by clicking the button and clicking the application in the popup.

Applications will automatically be started in an App Tab.

  • Install the experimental Web Apps extension. (You will need to confirm the installation)

Note: In this version, the list of installed applications used by the extension is not synchronized with the default (non-extension-based) list. After installing the extension, you will need to re-install any applications you already have.
