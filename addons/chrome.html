Mozilla : Web Applications for Chrome

Web Applications are applications that run on any device, and can be distributed through any store or directly by the developer.

An Experimental extension that builds support for Open Web Apps directly into Google Chrome.

Installing the chrome extension is a simple 3 step process:

1. Click to begin installation

2. Click continue to start the installation...

3. ...and click install to finish.

Launch

Once installed, you should go install some applications! Subsequent to that you can launch them directly from a button in your browser:

Or from the omnibox you can type app <space> [name] to quickly search for and launch applications:

Notes

The chrome extension is currently pre-alpha code and comes with a couple caveats:

  • Prompting at installation is not yet implemented -- applications will be silently installed. Prompting will be implemented as soon as the chrome folks implement infobars.
  • If you've already installed some apps into your browser before installing the chrome ext, you'll have to re-install, first run application migration will be implemented in a future release.

Finally, we welcome your contributions to this extension, either as code or comments in our mailing list.
