Open Web Apps for Chrome

An Experimental extension that builds support for Open Web Apps directly into Google Chrome.

Installation

Installing the chrome extension is a simple 3 step process:
  1. Click to begin installation
  2. Click continue to start the installation...
  3. ... and click install to finish.

Discover

We've also added experimental support for discovering which web sites provide an installable web app. The toolbar icon will glow and provide you with a prompt if the extension detects the presence of an installable application.

Note: Not all web applications support discovery, as we have not yet standardized on the mechanism.

Manage

In addition to being able to manage your applications from a dashboard of your choice, you can also manage them via 'about:apps'. Application manifests are also made available for inspection.

Extend

The Open Web Apps extension for Firefox is the ideal vehicle to test deeper desktop integration for web apps. To this end, we have added experimental support for propagating media key events (on Windows) to installed applications. With a simple one-line script, you can now control your favorite web-based music player via keyboard shortcuts even if Firefox is not the active window!

We are also working on many more experimental features to enhance the web app experience, such as integrating with Firefox Sync to seamlessly synchronize your installed apps across multiple devices.
